# Rendu de TP TP1 et TP2

## Devloppement local

Le code peut être lancé en local comme suit :
- Modifier la variable `severBaseUrl`dans TP1/script.js en `http://localhost:5000`
- Lancer `docker compose up`

## Déploiement

L'application est visible ici : https://archiapp-front-eu30.onrender.com/. Un délai d'initialisation peut être nécessaire le temps que les machines de render démarrent. En particulier, une erreur peut s'affichier dans le front le temps que le backend s'initialise. Il suffit d'attendre 30 à 40s pour que le site soit 100% fonctionnel.

Le backend peut être contacté ici : https://archiapp-front-eu30.onrender.com/
